# Xiyou Pack

Xiyou Pack 是 `ClawOS-Zhongtai` 的西游组织表达层。

它不是把产品做成取经玩梗，而是把一套多 Agent 组织系统，表达成：
- 有天命主线
- 有护行体系
- 有戒律与辨真
- 有路途调度
- 有法会、经卷、法器与因缘簿

的西行组织系统。

## 这套包适合什么
- 更有故事性与辨识度的主题组织产品
- 把 agent 角色写成护行、传经、辨真、巡路这类职责系统
- 在不改内核的前提下，换成更强叙事感的组织语言

## 当前结构
- 主位层：观音、女儿国主、黎山老母、金蝉使
- 部门层：传经台、照妖鉴司、行愿司、香积厨、戒律院、法器坊、仙缘簿、巡路司、法会司、经文馆
- 头像层：14 张 Pixar 风格西游头像
