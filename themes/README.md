# Themes

ClawOS-Zhongtai should support **one neutral kernel with multiple organization packs**.

The important distinction is:
- the **kernel** defines governance, routing, responsibility, and coordination logic
- a **theme pack** defines naming, role language, avatars, tone, and symbolic expression

That means users should eventually be able to choose between different organization packs without rewriting the underlying product logic.

## Current pack direction

### 1. `sansheng-pack`
The Three Departments / Six Ministries expression layer.

Best for:
- formal statecraft framing
- administrative hierarchy
- deliberation / review / dispatch clarity
- users who want a classical institutional language with direct role boundaries

### 2. `hougong-pack`
The inner-court / 后宫 expression layer.

Best for:
- ceremonial hierarchy
- rank, title, protocol, and power-language
- immersive female authority expression
- users who want stronger atmosphere without losing organizational clarity

## Long-term goal
A user should be able to:
- keep the same kernel
- switch the organization language
- swap avatars, role names, and tone
- preserve the same structural logic underneath

In short:

**same machine, different court language.**
