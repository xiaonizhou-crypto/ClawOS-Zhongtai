# Xianban Pack

Xianban Pack 是 `ClawOS-Zhongtai` 的仙班组织表达层。

它不是把产品做成玄幻门派，而是把一套多 Agent 组织系统，表达成：
- 有天规法度
- 有仙阶敕令
- 有各司其职的天庭官署
- 有文诰、天工、司命与法相庄严

的仙班组织系统。

## 这套包适合什么
- 更有天朝法度感与高级神话感的主题组织产品
- 把 agent 角色写成司诏、封驳、天宪、仙籍这类官署系统
- 在不改内核的前提下，换成更具仙威与秩序感的组织语言

## 当前结构
- 主位层：王母、上元主事仙妃、西华太真、少君承诏
- 部门层：诏命司、封驳司、都总司、天府库司、天宪司、天工司、仙籍司、值辰司、典仪司、玉章馆
- 头像层：准备接入仙班风格头像
