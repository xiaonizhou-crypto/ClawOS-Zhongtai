# Hougong Theme Pack

Hougong Theme Pack 是 `ClawOS-Zhongtai` 当前最完整的一套组织主题表达层之一。

它不是用来改中性治理内核的，
而是用来把一套多 Agent 组织系统，表达成：

- 有主位
- 有司处
- 有名分
- 有礼序
- 有压场感
- 有制度化女性权力语言

的后宫 / 内廷系统。

---

## 这套主题包里有什么

### 1. 角色文件
每个关键角色都按统一文件方式提供：
- `IDENTITY.md`
- `SOUL.md`

当前已覆盖：
- 皇后
- 贵妃
- 太后
- 东宫承旨
- 拟旨处
- 覆议处
- 掌事处
- 内务府
- 慎刑司
- 造办处
- 名册房
- 机宜处
- 规仪司
- 词馆

### 2. 统一规则文件
- `ROLE-MAPPING.md`
- `HIGH-COURT-MAPPING.md`
- `TONE-AND-TITLES.md`
- `AVATAR-ASSET-INDEX.md`
- `APPLY-TO-SANSHENG.md`
- `SANSHENG-MIGRATION-CHECKLIST.md`

### 3. 头像资源
头像位于：
- `assets/avatars/`

---

## 它适合做什么

这套主题包适合：
- 给现有三省六部项目换成后宫表达层
- 给 roster / dashboard / marketplace 做统一世界观表达
- 给 agent 角色写出统一人格、语气与体面感
- 做一套用户一秒入戏、但职责边界仍然清楚的组织皮肤

---

## 它不该做什么

这套主题包不应该：
- 重写内核逻辑
- 把主题语言写进业务判断条件
- 变成宫斗剧情包
- 为了氛围感牺牲结构清晰度

它的高级感来自：

**表达层极厚，内核层极稳。**

---

## 推荐阅读顺序

如果你第一次看这套主题包，建议按这个顺序读：
1. `ROLE-MAPPING.md`
2. `HIGH-COURT-MAPPING.md`
3. `TONE-AND-TITLES.md`
4. `AVATAR-ASSET-INDEX.md`
5. `APPLY-TO-SANSHENG.md`
6. `SANSHENG-MIGRATION-CHECKLIST.md`

然后再进入各角色目录看 `IDENTITY.md` / `SOUL.md`。
