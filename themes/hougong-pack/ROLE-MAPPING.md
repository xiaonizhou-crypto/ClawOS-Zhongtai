# Role Mapping

## 三省六部 / 现有组织角色 → 后宫组织语言映射

这份映射遵守一个原则：

**先按职责映射，再按世界观命名。**

也就是说，后宫表达层不是单纯换称呼，而是给同一套组织职责提供一层更有制度感、名分感与氛围感的语言。

---

## 核心中枢

- `zhongshu` / 中书省 → **拟旨处**
  - 职责：起草、谋划、把意图写成可执行章程

- `menxia` / 门下省 → **覆议处**
  - 职责：审看、封驳、检查完整性与风险

- `shangshu` / 尚书省 → **掌事处**
  - 职责：分派、调度、推进、收口执行链

---

## 资源 / 风险 / 制作 / 编制 / 调度

- `hubu` / 户部 → **内务府**
  - 职责：资源、账目、供给、轻重、节奏

- `xingbu` / 刑部 → **慎刑司**
  - 职责：纪律、边界、风险、禁区判断

- `gongbu` / 工部 → **造办处**
  - 职责：制作、搭建、实现、交付成品

- `libu-renli` / 吏部人力 → **名册房**
  - 职责：花名册、编制、位序、归属、人事秩序

- `bingbu` / 兵部 → **机宜处**
  - 职责：值守、应急、调度、支援、临机分派

---

## 可继续扩展的高阶角色

这些角色可以继续做，但当前还未正式落文件：

- `libu` / 礼部 → **规仪司**
  - 职责：规制、礼法、制度美感、对外体面

- `hanlinyuan` / 翰林院 → **词馆 / 内廷文馆**
  - 职责：文辞、诏令润色、文书典雅化、制度文本修饰

- `taizi` / 太子 → **东宫承旨 / 储位中枢**
  - 职责：承接上意、统摄未来主线、作为高阶继任或主轴角色

---

## 使用原则

1. 不要为了入戏牺牲结构清晰度。
2. 不要把后宫表达写成宫斗叙事。
3. 后宫主题应强调：
   - 名分
   - 权责
   - 调度
   - 规制
   - 值守
   - 供给
4. 主题层只负责组织语言，不替代中性治理内核。
