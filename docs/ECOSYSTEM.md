# Ecosystem

ClawOS-Zhongtai is the platform-facing repository for the AI organization kernel.

It is where the broader product direction lives:
- organization-first multi-agent design
- dashboard / roster / marketplace framing
- theme-layer philosophy
- long-term platform roadmap

## First landed theme pack

### 三省六部 / Three Departments · Six Ministries
The first theme-pack direction that has already been pushed far enough to look like a real runnable pack is:
- `clawos-pack-sansheng`

This pack represents:
- the first strong organizational language
- the first structured roster expression
- the first world where the kernel starts to feel staffed, ordered, and memorable

## Planned future worlds
- Journey to the West
- Palace systems
- Celestial courts
- more organizational worlds built on the same kernel

## Platform logic
The ecosystem works best when:
- Zhongtai explains the kernel
- each theme pack explains one world
- all packs feel related, but not collapsed into one messy repository

In short:

**one kernel, many worlds.**
